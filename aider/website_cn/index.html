功能特点

云端和本地大语言模型

Aider在Claude 3.7 Sonnet、DeepSeek R1与Chat V3、OpenAI o1、o3-mini和GPT-4o上表现最佳，但也可以连接到几乎任何大语言模型，包括本地模型。

代码库地图

Aider会为整个代码库创建地图， 帮助它在大型项目中更高效工作。

支持100多种编程语言

Aider支持大多数流行的编程语言： Python、JavaScript、Rust、Ruby、Go、C++、 PHP、HTML、CSS等更多语言。

Git集成

Aider自动提交更改并生成合理的提交信息。 使用熟悉的git工具轻松比较、管理和撤销AI的修改。

在你的IDE中使用

在你喜欢的IDE或编辑器中使用Aider。 只需在代码中添加注释请求更改， Aider就会开始工作。

图片和网页

在聊天中添加图片和网页，提供视觉上下文、 截图、参考文档等。

语音编程

与Aider语音交流代码！使用语音请求新功能、 测试用例或错误修复，让Aider实现这些更改。

代码检查和测试

Aider每次进行更改时自动检查和测试你的代码。 它能修复代码检查工具和测试套件发现的问题。

与网页聊天互通

Aider在LLM API上表现最佳， 但也可通过网页聊天界面工作。 Aider简化了与浏览器之间的代码复制粘贴过程。

开始使用

python -m pip install aider-install
aider-install

# 进入你的代码目录
cd /to/your/project

# DeepSeek
aider --model deepseek --api-key deepseek=<密钥>

# Claude 3.7 Sonnet
aider --model sonnet --api-key anthropic=<密钥>

# o3-mini
aider --model o3-mini --api-key openai=<密钥>
